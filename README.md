# ChemAmaze - Tamil Nadu State Board Syllabus Games

## Overview

ChemAmaze is a bilingual educational web application designed to provide interactive learning through game-based activities aligned with the Tamil Nadu State Board syllabus.
The platform enables students to access chapter-wise educational games in both English and Tamil languages. It currently supports Mathematics and Science subjects for Classes 4 to 8.

## Features

* 🌐 Bilingual Interface (English & Tamil)
* 📚 Subject Selection

  * Mathematics
  * Science
* 🎓 Class-wise Navigation (Classes 4–8)
* 📖 Chapter-wise Learning Games
* 🎮 Direct Integration with Wordwall Activities
* 📱 Responsive Design for Desktop and Mobile Devices
* 🏠 Easy Navigation with Back and Home Buttons
* 🎨 Clean and Student-Friendly User Interface

## How It Works

1. User enters the platform through the home page.
2. Selects preferred language (English or Tamil).
3. Chooses a subject.
4. Selects a class.
5. Views chapter-wise educational games.
6. Opens the corresponding Wordwall activity in a new tab.
